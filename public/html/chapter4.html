- There are two variations of this command: System.out.println() and System.out.print()

- If you write the System.out.println() command several times, then each time the text passed to it will be displayed on a new line. If System.out.print(), then the text will be printed on the same line. Example:

- A small remark. The println command does not print text on a new line — it prints text on the current line, but makes the next text appear on a new line.

- The println() command displays text and then adds a special invisible "line feed" character, so that the next text will be displayed at the beginning of a new line.
